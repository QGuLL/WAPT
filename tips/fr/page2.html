Un couple clé privée (.pem) - clé publique (.crt) sera généré pour permettre l'authentification des paquets.
Seuls les paquets signés de votre clé privée peuvent être installés sur les agents WAPT.
La clé privée (.pem) ne doit être dévoilée en aucun cas à une tierce partie.
Le certificat public (.crt) peut cependant être révélé. Il sera déployé sur les machines clientes via l'installeur personnalisé.

NB. : la clé privée ne doit être générée qu'une fois pour chaque organisation.
Pour plus d'informations : générer un certificat auto-signé.
