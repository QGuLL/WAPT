' L'installeur personnalisé WAPT (waptagent.exe) sert à intégrer votre certificat dans l'agent WAPT. L'agent WAPT doit être installé sur toutes les machines que vous souhaitez administer. Pour l'installer, vous pouvez utiliser plusieurs méthodes :  :
  • depuis une clé USB
  • depuis https://name_of_your_server/wapt/waptagent.exe
  • Déploiement_de_WAPT_par_GPO

Consultez la documentation sur Générer depuis la console WAPT
