Serveur WAPT

L'administration du serveur WAPT se fait à travers la console WAPT qui s'installe sur un poste Windows. Lors de l'installation du serveur WAPT sous Windows, la console est installée par défaut et peut être trouvé dans le menu "Démarrer".

Lors de l'installation du serveur WAPT sous Linux, il faut ensuite installer le client WAPT sur une machine d'administration et lancer la commande depuis le menu "Démarrer/Tous les programmes".

Pour intégrer un nouveau poste au serveur WAPT à la main, télécharger l'agent WAPT dans le menu à droite. La configuration de l'agent a été effectuée par le serveur et les paramètres par défaut sont les bons. Une fois le client WAPT installé, vous le retrouverez dans votre console.

Pour plus d'information, n'hésitez pas à vous reporter à la documentation sur le site dev.tranquil.it ou sur la mailing-list.
